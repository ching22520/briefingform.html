STRATEGY TOOLKIT　安裝教學

把客戶資料，
一鍵變成填好的 Brief

安裝 Brief Autofill 技能後，你只要把跟客戶往來的 email、LINE 對話、官網、文件丟給 Claude，它會自動整理成 Brief 並產出一條連結——策略企劃點開就能直接開工。以下四步，一次裝好永久可用。

整個安裝大約 3 分鐘，只需做一次

開始前，先準備兩樣

① 收到我傳給你的檔案 brief-autofill.zip（不用解壓縮，等等直接上傳這個 zip）。
② 使用 Claude 桌機版（Pro / Max 帳號），因為技能要靠「程式碼執行」才能運作。

1

打開設定，進入 Capabilities

在 Claude 桌機版左下角點你的頭像 → Settings（設定），左側選單點 Capabilities（不是 Customize，也不是 Claude Code——很多人會先找錯地方）。

3

上傳 brief-autofill.zip

開好上一步的開關後，在同一個 Capabilities 頁面繼續往下捲，會看到 Skills 清單（能看到你之前上傳過的技能，例如 proposal-strategy）。點右上角 「Add」 下拉選單 → 選 「Upload a skill」，把我給你的 brief-autofill.zip 選進去。上傳成功後，清單裡會多一行 brief-autofill。

裝好了！之後這樣用 👇

開一個新對話，把跟客戶往來的資料（email 內容、LINE 對話截圖、官網連結、文件檔）貼上或上傳，然後跟 Claude 說一句：

幫我把這些資料整理成 brief，產生填單連結 🙏 Claude好的，我先讀完你給的資料 → 整理出各欄位讓你核對 → 確認後給你一條「點開即填好」的連結，轉給策略企劃就能開工。

💡 Claude 會先列出它抓到的欄位讓你確認，缺的必填欄位也會標出來提醒你補——確認沒問題再產生連結就好。

常見問題

上傳後找不到技能有反應？
先確認第 2 步的「Cloud code execution and file creation」有開，再確認第 4 步的技能開關是開的。兩個都開了還不行，重開一個新對話再試。
手機版 App 也能用嗎？
技能不會跨裝置同步。如果你也想在手機用，要在手機版 App 的設定裡「另外再上傳一次」同一個 zip。
Claude 抓錯或漏抓怎麼辦？
直接在對話裡跟它說哪裡要改（例如「預算改成 300 萬量級」「這是比稿案」），它會更新後重新產生連結。
連結會把客戶資料傳到哪裡嗎？
不會。資料是編碼在網址本身，不會上傳到任何伺服器。連結給誰、誰才看得到。
